**C++ Exercises Bucket**

Este repositorio contiene una colección de ejercicios y fragmentos de código en C++ que he desarrollado. Mi objetivo es que sirvan como referencia personal para consultas rápidas y, al mismo tiempo, puedan ayudar a cualquier persona que esté aprendiendo el lenguaje.

**Información General**
A diferencia de otros repositorios estructurados, aquí encontrarás los ejercicios almacenados de forma directa. Cada archivo está pensado para resolver una duda puntual o demostrar una funcionalidad específica de C++.

¿Qué podrás encontrar aquí?
- Implementaciones de algoritmos básicos.

- Uso de la STL (Standard Template Library).

- Ejemplos de Programación Orientada a Objetos.

- Pruebas de lógica y estructuras de datos.

Para probar cualquiera de estos ejercicios, asegúrate de tener instalado un compilador como g++ (GCC) o clang.

**Cómo buscar**
Como los archivos no tienen un orden jerárquico, te recomiendo usar la función de búsqueda de GitHub (T en tu teclado dentro del repo) para localizar ejercicios por palabras clave en el nombre del archivo (ej. vector, clase, punteros).
